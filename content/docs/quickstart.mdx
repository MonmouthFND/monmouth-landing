## Connect to Monmouth

Monmouth exposes a standard Ethereum JSON-RPC API. Any EVM-compatible library works — ethers.js, viem, alloy, or raw HTTP.

```typescript
import { createPublicClient, http } from 'viem'

const client = createPublicClient({
  chain: {
    id: 7750,
    name: 'Monmouth',
    nativeCurrency: { name: 'ETH', symbol: 'ETH', decimals: 18 },
    rpcUrls: { default: { http: ['https://rpc.monmouth.io'] } },
  },
  transport: http(),
})

const blockNumber = await client.getBlockNumber()
```

## Query Capabilities

List the agent capabilities registered on the network:

```bash
curl -X POST https://rpc.monmouth.io \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "monmouth_listCapabilities",
    "params": [],
    "id": 1
  }'
```

Get details for a specific capability:

```bash
curl -X POST https://rpc.monmouth.io \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "monmouth_getCapabilitySchema",
    "params": ["swap"],
    "id": 1
  }'
```

## The Agent Loop

The core pattern for an LLM-powered agent on Monmouth:

```python
# 1. Read on-chain state
state = rpc.call("eth_call", [query_params])

# 2. LLM reasons about what to do
action = llm.plan(state, goal="swap 1 ETH for USDC")

# 3. Simulate the transaction
preview = rpc.call("eth_call", [action.to_tx()])

# 4. Submit with intent declaration
envelope = {
    "tx": action.to_signed_tx(),
    "intent": {
        "action": "swap",
        "expected_output": "~1800 USDC",
    }
}
tx_hash = rpc.call("eth_sendRawTransaction", [envelope])

# 5. Verify the outcome
receipt = rpc.call("eth_getTransactionReceipt", [tx_hash])

# 6. Repeat
```

## Configure Policies

Set up spending limits and operational boundaries using the policy engine:

```typescript
// Policy rules are registered per-agent via the Policy module
const policyRules = [
  {
    type: 'spending_cap',
    asset: 'ETH',
    maxPerTx: '1.0',
    maxPerDay: '10.0',
  },
  {
    type: 'rate_limit',
    maxTxPerMinute: 5,
  },
  {
    type: 'allowlist',
    contracts: ['0x...uniswap_router', '0x...aave_pool'],
  },
]
```

## Delegate a Session Key

Grant a scoped session key so your agent can transact without exposing the master key:

```typescript
// Session keys are scoped by time, spend limit, and allowed operations
const sessionGrant = {
  delegate: '0x...agent_hot_wallet',
  capabilities: ['swap', 'transfer'],
  expiresAt: Math.floor(Date.now() / 1000) + 3600, // 1 hour
  spendLimit: '5.0 ETH',
}
```

## Dual-VM: Target EVM or SVM

Monmouth runs EVM and SVM side by side. To target the Solana VM, prefix your transaction with the `0x4d` magic byte:

```typescript
// EVM transaction — standard Ethereum tx
const evmTx = await wallet.sendTransaction({ to: '0x...', value: parseEther('1') })

// SVM transaction — bincode-encoded Solana tx with 0x4d prefix
const svmTx = Buffer.concat([Buffer.from([0x4d]), solanaTxBytes])
const svmHash = await rpc.call("eth_sendRawTransaction", ['0x' + svmTx.toString('hex')])
```
